# FontMM - 字体打包工具

一款专为 **ColorOS 16** 设计的字体模块打包工具

## 🛠️ 使用步骤

1.  **准备模板**：下载提供的字体模块模板
2.  **上传字体**：上传必要的 `ch.ttf`（简体中文），可选上传 `tc.ttf`（繁体）及 `en.ttf`（英文数字）。
3.  **配置信息**：编辑 `module.prop` 模块配置文件（ID、名称、作者、版本等），支持表单与源码双模式。
4.  **打包生成**：一键生成 Magisk/KernelSU 兼容的模块压缩包。

## 🚀 本地开发

### 项目运行

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build
```
